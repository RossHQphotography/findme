# findme
A simple way for photographers to organise photos of clients and events!

## How to use
You will need your own hosting to use FindMe
You can find cheap webhosting at https://www.bisecthosting.com/web-hosting

### STEP 1 - SETUP
Download the source code

### STEP 2 - SETUP
Add the source code to your web hosting

### STEP 3 - REQUIRED SETUP
Open create.php and change the URL to your own hosting URL in this format:
\n https://YOUR_SITE_URL/user.php?id=
\n MAKE SURE TO INCLUDE /user.php?id=
\n EXAMPLE:     "url" => "https://rosshq.mcsite.ws/findme/user.php?id=" . $id

### STEP 4 - THE FUN PART
Take your photos
Give the QR to the client/subject 
They upload a selfie
You upload the photos into thier folder using /uploads/upload.php
